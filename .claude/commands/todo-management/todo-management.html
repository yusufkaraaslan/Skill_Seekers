/add-to-todos Prompt

/add-to-todos [optional description]

Captures the current conversation context as a structured todo item in TO-DOS.md.

Works anywhere you use Claude Code. Each project gets its own todo list automatically.

  • The problem or task - What needs to be done and why
  • File paths with line numbers - Exact locations (auth.ts:145-230)
  • Technical context - Error messages, root causes, constraints
  • Timestamp - When you captured this idea
  • Solution hints - Approach ideas if you've already discussed them (optional)

Each todo is self-contained with enough context for Claude to understand the task weeks later.

/check-todos Prompt

/check-todos

Lists all todos in a compact numbered format. Select one to load full context and begin work.

Automatically removes the todo when you select it and checks for project-specific workflows.

  • Parse all todos - Shows compact list with dates
  • You select a number - Pick what you want to work on
  • Load full context - Problem, Files, Solution hints all loaded
  • Check for workflows - Reads CLAUDE.md and .claude/skills/ to see if this should use a specific workflow
  • Suggest skill if found - "This looks like MCP work, use the create-mcp-servers skill?"
  • Remove from list - Todo is deleted, you're ready to work
