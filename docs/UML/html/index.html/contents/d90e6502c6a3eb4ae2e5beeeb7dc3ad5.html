SubcommandParser

UMLClass
Skill Seekers :: skill_seekers :: Parsers :: SubcommandParser

Description

Abstract base for CLI subcommand argument parsers. Defines register(subparsers) to add a subcommand with its arguments. 30+ implementations, one per CLI subcommand.
